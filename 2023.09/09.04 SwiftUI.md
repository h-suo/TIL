## SwiftUI

### SwiftUI로 간단한 UI 그리기
```swift
struct MyImageView: View {
    var body: some View {
        VStack {
            Image("Me")
                .clipShape(Circle())
                .overlay {
                    Circle().stroke(.white, lineWidth: 1)
                }
            .shadow(radius: 10)
            Text("Hi, my name is Erick")
                .font(.title)
                .foregroundColor(.gray)
        }
    }
}

struct MyImageView_Previews: PreviewProvider {
    static var previews: some View {
        MyImageView()
    }
}
```
- `VStack` 내에 `Image`와 `Text` 배치
