## mutating
- 구조체에서 함수가 내부 프로퍼티를 수정하기 위해서 필요함
- 보통 값 타입의 프로퍼티는 인스턴스 메서드에서 수정 불가
```swift
struct User {
  var age: Int = 10

  mutating func getOld() {
    age += 10
  }
}
```

## 짝프로그래밍&README
- 혼자 하는게 편할 수는 있어도 팀과 같이 하는게 더 빠른 성장을 가져다 줄 수도 있다.
- 단순히 작성하는 것이 아닌 남이 보기 편하고 쉬운 것이 좋다.
